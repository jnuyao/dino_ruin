# 项目总览

## 为什么做这个项目
6-7 岁儿童需要高反馈、低门槛、可重复的学习体验。传统题目页文字负担重，容易失去兴趣。本项目将数学训练包装为遗迹闯关游戏，核心目标是“孩子能自己玩下去”。

## 目标用户
- 主要用户：6-7 岁儿童
- 次要用户：家长（进度查看、重置存档）

## MVP 范围
- 地图选关（20 关顺序解锁）
- 3 类关卡交互
- 提示系统（最多 3 条）
- 碎片奖励 + 孵蛋 + 图鉴
- 本地存档

## 当前状态
- 已完成可玩闭环
- 已完成游戏化 UI 和通关弹层
- 已具备后续接入云存档的 API 骨架

## 成功标准
- 孩子可在少引导情况下完成前 5 关
- 全 20 关可通关
- 刷新后进度保留
